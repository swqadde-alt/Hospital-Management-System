# Hospital Management System

## Description

This project is a console-based **Hospital Management System** developed in C as part of my programming learning journey. It allows administrators and patients to manage hospital operations through a menu-driven interface by registering patients, managing doctors, scheduling appointments, canceling appointments, and viewing hospital records.

While developing this project, I focused on improving my understanding of structures, functions, arrays, modular programming, input validation, and data management. The project demonstrates the practical application of fundamental C programming concepts while emphasizing clean, readable, and well-organized code. Compared to my previous projects, it reflects significant progress in both software design and problem-solving skills.

## Features

* Administrator login system
* Doctor management
* Patient registration
* Appointment scheduling
* Appointment cancellation
* View doctors, patients, and appointments
* Input validation
* Menu-driven interface
